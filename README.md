# Eigene Zigbee-Geräte mit ESP32 entwickeln

Dieses Repository enthält die **Codebeispiele und Projekte** zum Kurs und Buch rund um Zigbee mit dem ESP32, Home Assistant und Zigbee2MQTT.

Kurs: https://www.pixeledi.eu/akademie/zigbee-esp32

## Inhalte

* **Zigbee Grundlagen:** Coordinator, Router, End Device, Endpoints und Cluster
* **Praxis mit ESP32:** Sensoren, Schalter, Türkontakte und Aktoren
* **Tooling:** Arduino IDE, PlatformIO, MQTT, Zigbee2MQTT und Home Assistant
* **Debugging:** Typische Fehler erkennen und systematisch lösen
* **Betrieb:** Deep Sleep, Batteriebetrieb und stabile Mesh-Strukturen

## Ziel des Repos

Der Fokus liegt auf praxisnahen, nachvollziehbaren Beispielen. So kannst du die Projekte aus dem Kurs direkt nachbauen und eigene Zigbee-Geräte selbst weiterentwickeln.

## Für wen

* Maker mit Interesse an Zigbee und Smart Home
* Arduino- und ESP32-Nutzer
* Alle, die eigene Zigbee-Geräte technisch sauber umsetzen möchten
